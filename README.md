# Реферальная программа «Рабочие Руки»

Репозиторий с материалами по проекту реферальной программы для платформы «Рабочие Руки»: техническое задание, кликабельные прототипы интерфейса (мобильное приложение работника + CRM администратора) и аналитика расхождений по условиям программы.

## О проекте

Реферальная программа позволяет действующим работникам платформы приглашать новых исполнителей по персональной ссылке и получать вознаграждение за их активность. В составе:

- **Мобильное приложение работника** — экран «Пригласить друга», личная реферальная ссылка, статус приглашённых, история выплат.
- **CRM администратора** — управление программой, ставки, модерация рефералов, выплаты, отчётность.
- **Уведомления** — push/SMS/in-app в ключевых точках воронки (регистрация реферала, первая смена, выплата).

Прототипы кликабельные, реализованы единым HTML-файлом без сборки и зависимостей — открываются в браузере двойным кликом.

## Как открыть прототип на локальной машине

### Способ 1 — двойной клик (самый простой)

1. Склонируйте репозиторий или скачайте ZIP:
   ```bash
   git clone https://github.com/azamdoctor/REF_RR.git
   cd REF_RR
   ```
2. Откройте `prototype_v3.html` двойным кликом — он запустится в системном браузере. Никакая сборка и сервер не нужны.

### Способ 2 — через локальный HTTP-сервер (рекомендуется)

Некоторые браузеры из соображений безопасности блокируют часть функциональности при открытии HTML через `file://`. Чтобы избежать сюрпризов, поднимите локальный сервер.

**Через Python (есть на macOS и Linux из коробки):**
```bash
cd REF_RR
python3 -m http.server 8000
```
Затем откройте: <http://localhost:8000/prototype_v3.html>

**Через Node.js (если установлен):**
```bash
cd REF_RR
npx serve .
```
Адрес покажется в терминале (обычно <http://localhost:3000>).

**Через VS Code:**
Установите расширение **Live Server**, откройте папку, кликните правой кнопкой на `prototype_v3.html` → *Open with Live Server*.

### Остановить сервер
В терминале, где запущен сервер, нажмите `Ctrl+C`.

## Навигация по прототипу v3

После открытия `prototype_v3.html`:

- **Слева** — мобильные экраны работника (5 экранов): главный экран программы, моя ссылка, мои рефералы, история выплат, детали реферала.
- **Справа** — CRM администратора (4 вкладки): дашборд, рефералы, выплаты, настройки ставок.
- **Уведомления** — всплывающие карточки в верхней части мобильных экранов.
